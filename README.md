# 💬 Chirp – Real-Time Chat Application

A modern, production-ready real-time chat application inspired by WhatsApp and Telegram. Built using **React**, **Vite**, **TypeScript**, **Node.js**, **Express**, **PostgreSQL**, and **Socket.IO**, PulseChat delivers fast, secure, and seamless one-to-one communication with support for media sharing and voice messages.

> Designed with scalable architecture, clean code practices, and a premium user experience.

---

## 🚀 Features

- 🔐 Secure JWT Authentication
- 💬 Real-Time One-to-One Messaging
- ✅ Sent, Delivered & Seen Message Status
- 🖼️ Image Upload, Preview & Download
- 🎙️ Voice Message Recording & Playback
- 😀 Emoji Support
- ⌨️ Typing Indicators
- 🟢 Online / Offline Presence
- 📱 Responsive Design
- ⚡ Socket.IO Real-Time Communication
- 🗄️ PostgreSQL Database
- 🎨 Modern & Clean UI

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Harikarthick19/Chat-Application.git
cd Chat-Application
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
